# PyNode

Docker image ready for Python and NodeJS development.

With this image, you can:

* Install pip packages:

    
    RUN pip3 install requirements.txt
    

* Install Node packages (with NPM):


    npm install
    
* Install Node packages (with YARN):
    
    
    yarn install
